# 💬 Smart Chat

A modern real-time chat application built using **Flutter** and **Firebase**, enabling secure one-to-one and group communication with a clean and user-friendly interface.

---

## 📌 Features

### 🔐 Authentication
- User Registration
- User Login
- Forgot Password
- Secure Firebase Authentication
- Logout

### 💬 Chat
- Real-time One-to-One Messaging
- Real-time Group Chat
- Image Sharing
- Emoji Picker
- Typing Indicator
- Read Receipts
- Message Deletion
- Online/Offline Status
- Last Seen

### 👤 User Management
- View Profile
- Edit Profile
- Search Users

### ⚙️ Settings
- Edit Profile
- Change Password
- Notification Toggle (UI)
- About
- Logout

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/srinidhi829/SMART_CHAT_APP.git
```

### Navigate to the Project

```bash
cd SMART_CHAT_APP
```

### Install Dependencies

```bash
flutter pub get
```

### Run the Application

```bash
flutter run
```
